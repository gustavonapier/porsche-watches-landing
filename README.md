# ⌚ Porsche Watches - Landing Page

Este projeto é uma **landing page moderna** inspirada no design elegante e minimalista da Porsche.  
O objetivo foi praticar conceitos de **desenvolvimento front-end**, focando em layout, responsividade e construção de interfaces.

---

## 🚀 Demonstração

A página apresenta um produto em destaque com um layout moderno dividido em duas cores, além de uma página de **criação de conta (Sign Up)**.

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de **praticar desenvolvimento front-end**, principalmente:
